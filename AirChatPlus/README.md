# AirChatPlus v5 – 完整增强版

## 功能清单
1. **启动弹窗**：确认插件注入成功。
2. **显示访客总数**：在帖子详情页顶部添加蓝色标签。
3. **拦截权限弹窗**：用自己的提示框替代系统弹窗。
4. **排行榜自动翻页拼接**：自动请求所有页面，合并数据，突破100条限制。

## 编译
```bash
make
```

## 注入
使用轻松签等工具将 `AirChatPlus.dylib` 注入 `AirChat.ipa`，重签名后安装。

## 验证
- 打开 AirChat，看到弹窗即注入成功
- 帖子详情页顶部出现蓝色访客人数标签
- 点击「访客数据」，弹出我们自己的提示框
- 排行榜控制台显示翻页合并日志

## 风险提醒
- **排行榜翻页会连续发起多次请求**，可能触发服务端风控，**建议用小号测试**
- 大量数据渲染可能导致低端机型卡顿
- 访客具体列表仍**无法查看**（服务端硬性权限）
