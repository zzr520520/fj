# AirChatPlus – 飞行圈增强插件

本插件是一个动态库（dylib），可注入到 AirChat 应用中，实现以下功能：

- **强制显示访客数量**：无论是否为作者，帖子卡片和详情页都会显示浏览数和曝光数（若服务端返回）。
- **显示访客入口**：所有帖子都会显示"访客"区域，但非作者点击时会提示"仅作者可查看"，避免请求失败。
- **突破排行榜显示上限**：自动将排行榜请求的 `pageSize` 参数改为 200，尝试显示更多排名（需服务端支持）。

## 编译

在 Mac 终端中执行：

```bash
make
```

将生成 `AirChatPlus.dylib`。

## 注入方法

1. 使用轻松签（或类似工具）将 `AirChatPlus.dylib` 注入到 `AirChat.ipa` 中。
2. 使用您的证书重签名 IPA。
3. 安装到设备即可生效。

## 注意事项

- 本插件不会修改任何服务端数据，仅调整客户端显示和请求参数。
- 访客列表仍受服务端权限控制，非作者无法查看具体名单。
- 若 AirChat 版本更新导致类名或方法名改变，需重新适配（使用 class-dump 或 FLEX 获取最新信息）。
- 编译时需确保 Xcode 命令行工具已安装，且 iOS SDK 路径正确。
