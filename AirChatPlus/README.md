# AirChatPlus v3 – 稳定版

本插件专注于：
1. **显示访客人数**：在帖子详情页顶部添加蓝色标签，显示 `👀 访客人数: xxx`。
2. **排行榜突破**：自动将包含 `rank` 的请求中的 `pageSize` 修改为 500。

## 与之前版本区别

- **不再 Hook 任何业务类**，仅 Hook 系统类（`UIViewController` 和 `NSURLSession`），避免因类名变化导致的崩溃。
- **只添加新 UI**，不修改或隐藏任何原有视图。

## 编译

```bash
make
```

## 注入

使用轻松签等工具将 `AirChatPlus.dylib` 注入 `AirChat.ipa`，重签名后安装。

## 验证

- 打开 AirChat，立即弹出"已注入"提示
- 进入任意帖子详情页，如果服务端返回了访客数量，顶部会出现蓝色标签
- 排行榜请求会被修改（查看控制台日志确认）

## 故障排除

- 没有弹窗：检查 dylib 是否成功注入，签名是否有效
- 标签未出现：服务端可能未返回 `visitorCount` 或 `viewCount` 字段，或详情页类名不匹配
- 排行榜仍只显示 100：服务端硬性限制了最大返回数
