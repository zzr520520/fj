# AirChatPlus v2 – 飞行圈增强插件（通用版）

本版本采用**动态类查找**和**多方法尝试**，兼容性更强，并增加了**启动弹窗**和**详细日志**，方便验证是否生效。

## 新特性

- 启动时弹窗确认注入成功
- 自动搜索包含 `Moment`、`Cell`、`Detail` 的类并尝试 Hook
- 强制显示访客数量（如果数据存在）
- 拦截排行榜请求，将 `pageSize` 改为 500
- 所有操作输出 `NSLog`，可用设备控制台查看

## 编译

```bash
make
```

## 注入

使用轻松签等工具注入到 AirChat.ipa 并重签名，安装到设备。

## 验证

- 打开 AirChat，应立刻弹出提示框
- 查看飞行圈帖子，若原帖有访客数据，应能显示数量
- 点击「访客数据」，非作者会弹窗提示并显示已有数量
- 排行榜请求 URL 会被修改（可在控制台查看日志）

## 调试

连接设备，打开 Xcode -> Window -> Devices and Simulators，选择设备，查看 Console 日志，过滤 `AirChatPlus` 即可。

## 常见问题

- 没有弹窗：可能是注入失败或签名问题
- 功能不生效：应用版本可能更新，类名变化。使用 class-dump 或 FLEX 获取最新类名
- 排行榜仍只显示 100 条：服务端可能硬性限制
