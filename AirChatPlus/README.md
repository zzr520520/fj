# AirChatPlus v4 – 最终版

## 功能

1. **显示访客人数**：在帖子详情页顶部添加蓝色标签，显示 `👀 访客人数: xxx`
2. **拦截权限弹窗**：点击"访客数据"按钮时，用自己的弹窗取代系统弹窗（"仅作者可查看"/"开通SVIP"），显示已有访客数量
3. **排行榜突破**：修改包含 `rank` 的请求 `pageSize` 为 500

## 安全性

- **仅 Hook 系统类**：`UIViewController`、`UIButton`、`NSURLSession`
- **不触碰任何业务类**，不会导致崩溃或空白页
- **只添加新 UI，不修改原有视图**

## 为什么无法突破访客列表权限？

- `/interaction/visitor` 接口强制校验**作者身份 + VIP 等级**，纯服务端校验
- 客户端无法伪造作者身份（需登录态 + userId 匹配）
- 本插件只能显示服务端已返回的 `visitorCount`/`viewCount` 数值

## 编译

```bash
make
```

## 注入

使用轻松签等工具将 `AirChatPlus.dylib` 注入 `AirChat.ipa`，重签名后安装。

## 验证

- 打开 AirChat，立即弹出"已加载"提示
- 帖子详情页顶部出现蓝色访客人数标签（若服务端返回了数据）
- 点击"访客数据"，弹出我们的提示框而非系统权限提示
- 控制台日志过滤 `AirChatPlus` 可看到详细 Hook 信息
